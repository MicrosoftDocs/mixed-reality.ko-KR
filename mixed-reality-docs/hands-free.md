## <a name="transitioning-in-and-out-of-hands-freey"></a>실습 freey 내부 및 외부로 전환

응용 프로그램의 요구 사항을 유지, 음성 명령, 또는 단일 음성 명령을 사용 하 여 여부를 항상 사용 되도록 핸 즈 프리가, "select" 경우 UI에 적절 한 객실을 확인 해야 합니다.
