# <a name="environment-considerations-for-hololens"></a>HoloLens 환경 고려 사항

HoloLens는 holographic을 "실제" 세계와 혼합 하 여 holograms를 주변에 배치 합니다. Holographic 앱 창이 벽면에서 "중단" 되 고, holographic ballerina가 토끼 친구의 머리 위에 있는 윗면, 자신도 모르는 귀 sit에서 회전 합니다. 몰입 형 게임 또는 앱을 사용 하는 경우 holographic 세계를 대상으로 하는 환경이 확산 될 것입니다. 하지만 계속 해 서 공간을 볼 수 있습니다.

장치를 끄는 경우에도 배치한 위치를 유지 하는 holograms 있습니다. 

## <a name="setting-up-an-environment"></a>환경 설정

HoloLens 장치에서는 사용자를 공간으로 *추적* 하 여 안정적이 고 정확한 holograms을 저장 하는 방법을 알고 있습니다. 적절 한 추적을 사용 하지 않으면 장치에서 환경이 나 사용자를 인식 하지 못합니다. 따라서 holograms는 잘못 된 위치에 표시 되거나 매번 동일한 위치에 표시 되지 않거나 전혀 표시 되지 않을 수 있습니다.

추적 성능은 사용자가 있는 환경에 의해 크게 영향을 받으며 안정적이 고 일관 된 추적을 유도 하는 환경을 조정 하는 것은 과학이 아니라 아트입니다. 여러 가지 환경 요소를 함께 사용 하 여 추적을 가능 하 게 하 고, 혼합 현실 개발자는 추적을 개선 하기 위해 공간을 조정할 수 있는 몇 가지 요인을 고려해 야 합니다.
 
### <a name="lighting"></a>조명
Windows Mixed Reality에서는 시각적 표시를 사용 하 여 사용자의 위치를 추적 합니다. 환경이 너무 밝습니다 경우 카메라는 포화 상태가 되 고 아무 것도 표시 되지 않습니다. 환경이 너무 어둡고 카메라에서 충분 한 정보를 선택할 수 없으며 아무것도 표시 되지 않습니다. 광원은 사용자가 활동 없이 볼 수 있는 것은 물론 밝은 색은 매우 밝습니다.

전체 dim 영역에서 밝은 빛의 점이 있는 영역에는 카메라에서 밝은 공간으로 이동 하는 경우를 조정 해야 하기 때문에 문제가 발생 합니다. 이로 인해 장치는 "손실" 될 수 있으며, 광원의 변경 내용이 위치 변경과 동일 하다 고 생각할 수 있습니다. 영역에서 안정화 된 밝은 수준으로 인해 추적이 향상 됩니다.

모든 실외 조명은 시간이 지남에 따라 현저 하 게 다를 수 있으므로 추적기에서 불안정성을 일으킬 수도 있습니다. 예를 들어 여름 및 겨울에 있는 동일한 공간의 추적으로 인해 secondhand의 외부에 있는 조명이 서로 다른 시간에 더 높을 수 있기 때문에 매우 다른 결과가 생성 될 수 있습니다.

Luxmeter 있는 경우 안정적인 500-1000 lux를 시작 하는 것이 좋습니다. 

#### <a name="types-of-lighting"></a>광원 유형
공간에 있는 다른 유형의 빛도 추적에 영향을 줄 수 있습니다. Ac 전기를 통해 실행 되는 AC 전기를 사용 하는 전구 펄스-AC 주파수가 50Hz 이면 조명이 50Hz로 펄스 됩니다. 사용자의 경우에는이 펄스를 알 수 없습니다. 그러나 HoloLens의 30fps 카메라는 이러한 변경 내용을 볼 수 있습니다. 일부 프레임은 제대로 작동 하지 않으며, 일부 프레임은 제대로 작동 하지 않으며, 일부는 카메라에서 밝은 펄스를 보정 하려고 할 때 과도 하 게 노출 됩니다.

미국에서 전기 주파수 표준은 60Hz 이므로 전구를 사용 하는 전구의 프레임 속도는 hololens의 프레임 속도에 맞게 harmonized 됩니다. 그러나 대부분의 국가에는 50Hz의 AC 주파수 표준이 있습니다. 즉, 일부 Hololens 프레임은 펄스 중에 수행 되 고 다른 항목은 사용 되지 않습니다. 특히 유럽의 형광등 조명에서 문제가 발생 하는 것으로 알려져 있습니다. 

깜빡임 문제를 해결 하기 위해 몇 가지 작업을 시도할 수 있습니다. 온도, 전구 사용 기간 및 준비 주기는 형광등의 일반적인 원인 이며 전구을 대체 하는 데 도움이 될 수 있습니다. 전구를 강화 하 고 현재 그리기를 일정 하 게 하는 것도 도움이 될 수 있습니다. 

### <a name="items-in-a-space"></a>공간의 항목
HoloLens는 *기능*이라고 하는 고유한 환경 랜드마크를 사용 하 여 공간에서 자신을 찾습니다. 

장치는 공간 위치를 알 수 없기 때문에 기능 저하 영역에서 장치를 거의 추적 하지 않을 수 있습니다. 공간 벽에 기능을 추가 하는 것은 일반적으로 추적을 개선 하는 좋은 방법입니다. 포스터, 기호는 벽, 공장, 고유 개체 또는 기타 유사한 항목에 대 한 모든 도움을 taped. 매우 복잡 한 책상은 좋은 추적으로 이어지는 환경의 좋은 예입니다. 단일 영역에는 다양 한 기능이 있습니다. 

또한 동일한 공간에 고유한 기능을 사용 합니다. 예를 들어 동일한 포스터는 벽에서 여러 번 반복 되는 경우에는 HoloLens가 표시 하는 반복적인 포스터의 수를 알 수 없기 때문에 장치 혼란이 발생 합니다. 고유한 기능을 추가 하는 일반적인 방법 중 하나는 마스킹 테이프의 줄을 사용 하 여 공간의 벽과 바닥을 따라 고유한 nonrepetitve 패턴을 만드는 것입니다. 

사용자에 게 질문 하는 것은 좋은 질문입니다. 장면을 약간만 볼 경우 공간에서 고유한 사용자를 찾을 수 있나요? 그렇지 않은 경우 장치에도 문제를 추적 하는 것입니다.

#### <a name="wormholes"></a>Wormholes
동일 하 게 보이는 영역 또는 지역이 두 개인 경우 tracker는 동일 하다 고 생각할 수 있습니다. 이로 인해 장치는 다른 위치에 속여 것으로 간주 됩니다. 이러한 유형의 반복 영역 *wormholes*을 호출 합니다. 

Wormholes을 방지 하려면 동일한 공간에서 동일한 영역을 방지 합니다. 때로는 동일한 영역에 공장 스테이션, 건물의 windows, 서버 랙 또는 작업 스테이션이 포함 될 수 있습니다. Labelling 영역을 추가 하거나 각 유사한 영역에 고유한 기능을 추가 하 여 wormholes를 완화할 수 있습니다.
 
### <a name="temporal-stability-of-a-space"></a>공간의 임시 안정성
환경이 지속적으로 이동 하 고 변경 되는 경우 장치에는 찾을 안정적인 기능이 없습니다. 

사용자를 포함 하 여 공간에 있는 더 많은 이동 개체가 추적을 잃지 않는 것이 더 쉽습니다. 컨베이어 벨트 이동, 다른 생성 상태의 항목 및 공간에 있는 많은 사용자는 추적 문제를 야기 하는 것으로 알려져 있습니다.
 
### <a name="proximity-of-the-user-to-items-in-the-space"></a>사용자가 공간에 있는 항목에 근접
눈에 잘 가까운 개체에 대 한 관심을 겪던 수 없는 경우와 마찬가지로, 개체가 카메라에 가까이 있는 경우 HoloLens를 합니다. 두 카메라에서 개체가 너무 가깝거나 개체가 단일 카메라를 차단 하는 경우에는 해당 장치에 개체에 대 한 추적 문제가 훨씬 더 많이 발생 합니다. 

카메라는 개체에서 15cm 보다 더 가까운 것을 볼 수 없습니다.
 
### <a name="surfaces-in-a-space"></a>공간의 표면
강력한 반사 표면은 추적에 영향을 주는 각도에 따라 달라질 수 있습니다. 새 자동차를 생각해 보겠습니다. 이동 하는 경우 조명이 반영 되 고 이동 하는 동안 표면에 다른 개체가 표시 됩니다. 추적기로 화면에 반영 되는 다른 개체는 변화 하는 환경을 나타내며 장치는 추적을 상실 합니다.

Less 개체는 추적 하기가 더 쉽습니다.

### <a name="wifi-fingerprint-considerations"></a>WiFi 지문 고려 사항
WiFi를 사용 하도록 설정 하면 실제 WiFi 네트워크/라우터에 연결 되지 않은 경우에도 지도 데이터는 WiFi 지문과 상관 관계가 지정 됩니다. WiFi 정보를 사용 하지 않으면 공백과 holograms을 인식 하는 데 약간 더 느릴 수 있습니다. WiFi 신호가 현저 하 게 변경 되 면 장치는 다른 공간에 있는 것으로 생각할 수 있습니다.

네트워크 id (예: SSID, MAC 주소)는 Microsoft로 전송 되지 않으며 모든 WiFi 참조는 HoloLens에서 로컬로 유지 됩니다.

## <a name="mapping-new-spaces"></a>새 공백 매핑
새 공간을 입력 하거나 기존 공간을 로드 하는 경우 공간에 분산 된 메시 그래픽이 표시 됩니다. 이는 장치가 사용자 환경을 [매핑하](spatial-mapping-design.md)는 것을 의미 합니다. 

Holograms를 배치 하는 데 문제가 있는 경우 HoloLens가 더 완전 하 게 매핑할 수 있도록 공간을 탐색 해 보세요. 

HoloLens가 공간을 매핑할 수 없거나 보정을 벗어난 경우 제한 된 모드로 전환 될 수 있습니다. 제한 된 모드에서는 holograms를 주변에 놓을 수 없습니다.

## <a name="environment-management"></a>환경 관리
사용자가 "정리" 하는 데 사용할 수 있는 두 가지 설정이 있으며,이로 인해 holograms에서 공간을 "잊어버린" 상태로 만듭니다.  이러한 항목은 설정 앱의 "Holograms and environment"에 있고, 두 번째 설정은 설정 앱의 "개인 정보" 아래에도 표시 됩니다.

1.  주변 holograms 삭제 –이 설정을 선택 하 여 HoloLens는 장치가 있는 "현재 공간"에 대해 고정 된 모든 holograms 및 저장 된 모든 맵 데이터를 지웁니다.  Holograms가 다시 동일한 공간에 배치 되 면 새 맵 섹션이 만들어지고 해당 위치에 대 한 데이터베이스에 저장 됩니다.

2.  모든 holograms 삭제 –이 설정을 선택 하 여 HoloLens는 전체 공간 데이터베이스에서 모든 지도 데이터와 고정 holograms를 지웁니다.  Holograms는 시이 없으며, 데이터베이스에 맵 섹션을 다시 저장 하기 위해 holograms를 새로 배치 해야 합니다.

### <a name="managing-your-spaces"></a>공간 관리

지도 섹션과 다른 공백이 HoloLens 장치에 로컬로 저장 된 단일 데이터베이스로 축소 되었습니다. 지도 데이터베이스는 내부 시스템에만 액세스할 수 있는 상태로 안전 하 게 저장 되며, PC에 연결 되어 있거나 파일 탐색기 앱을 사용 하는 경우에도 장치 사용자에 게는 액세스할 수 없습니다. Bitlocker를 사용 하도록 설정 하면 저장 된 맵 데이터도 암호화 됩니다.

Holograms가 위치/holograms 사이에 연결 경로가 없는 다른 위치에 배치 되는 경우 여러 개의 지도 구성 요소가 있습니다.  동일한 지도 섹션 내에서 고정 된 Holograms는 현재 공간에서 "주변"으로 간주 됩니다.

공유 홀로그램 시나리오를 사용할 수 있도록 "현재 공간" (현재 인식 된 지도 구성 요소의 일부)의 작은 하위 집합을 내보내는 개발자 API가 있습니다.  현재 매핑된 모든 공간의 전체 데이터베이스를 다운로드 하는 메커니즘은 없습니다.


## <a name="hologram-quality"></a>홀로그램 품질

Holograms는 환경 전체 (높음, 낮음, 모든 사용자에 게 배치 될 수 있지만, 눈에 [holographic 프레임](holographic-frame.md) 을 통해 볼 수 있습니다. 최상의 보기를 얻으려면 전체 프레임을 볼 수 있도록 장치를 조정 해야 합니다. 사용자 환경에 대 한 탐색을 주저 하지 말고 탐색 하세요!

[Holograms](hologram.md) 가 명확 하 고 명확 하며 안정적으로 보이도록 HoloLens를 보정 해야 합니다. HoloLens를 처음 설정 하는 경우이 프로세스를 안내 합니다. 나중에 holograms가 표시 되지 않거나 많은 오류가 표시 되는 경우 조정을 수행할 수 있습니다.

### <a name="calibration"></a>보정

Holograms 또는 흔들리는가 나 holograms를 배치 하는 데 문제가 있는 경우 먼저 [보정 앱](calibration.md)을 사용해 보세요. 이 앱은 HoloLens를 사용 하는 동안 discomfort 발생 하는 경우에도 도움이 됩니다.

보정 앱에 가져오려면 설정 > 시스템 > 유틸리티로 이동 합니다. 보정 열기를 선택 하 고 지침을 따릅니다.

보정 앱을 실행 중이 고 홀로그램 품질이 여전히 발생 하 고 있거나 "손실 추적" 메시지가 자주 표시 되는 경우 센서 튜닝 앱을 사용해 보세요. 설정 > 시스템 > 유틸리티로 이동 하 여 센서 튜닝 열기를 선택 하 고 지침을 따릅니다.

다른 사람이 HoloLens를 사용 하려는 경우 먼저 보정 앱을 실행 해야 장치가 적절 하 게 설정 됩니다.
