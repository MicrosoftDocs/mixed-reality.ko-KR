# <a name="advanced-hololens-emulator-and-mixed-reality-simulator-input"></a>고급 HoloLens 에뮬레이터 및 혼합 현실 시뮬레이터 입력

대부분의 에뮬레이터 사용자에 대 한 기본 입력된 컨트롤을 사용 해야만 합니다 [HoloLens 에뮬레이터](using-the-hololens-emulator.md#basic-emulator-input) 또는 [Windows Mixed Reality 시뮬레이터](using-the-windows-mixed-reality-simulator.md#basic-simulator-input)합니다. 아래 세부 정보는 더 복잡 한 유형의 입력을 시뮬레이션 해야 찾을 수 있는 고급 사용자입니다.

> [!NOTE]
> HoloLens 2 관련 된 자세한 지침 [예정](index.md#news-and-notes)합니다.

## <a name="concepts"></a>개념

HoloLens 에뮬레이터와 Windows Mixed Reality 시뮬레이터에 대 한 가상 입력 제어 시작 하려면 먼저 몇 가지 개념을 이해 해야 합니다.

동작을 제어 하 고 위치 및 장면에서의 방향을 변경 가리킵니다. 대상된 제어할 수 있는 개체에 대 한 동작 회전 및 3 개의 축 따라 변환 (이동)를 사용 하 여 제어 됩니다.
* **Yaw**: 왼쪽 또는 오른쪽 설정 합니다.
* **피치**: 위로 또는 아래로 설정 합니다.
* **롤백**: 나란히를 배포 합니다.
* **X**: 왼쪽 또는 오른쪽 이동 합니다.
* **Y**: 위 또는 아래로 이동 합니다.
* **Z**: 앞으로 또는 뒤로 이동 합니다.

[제스처](gestures.md) 및 동작 컨트롤러 입력 방법을 밀접 하 게 매핑되는 물리적 장치:
* **작업**: Thumb 엄지와 누르거나 컨트롤러에서 실행 단추 끌어오기의 동작을 시뮬레이션 합니다. 예를 들어, 작업 입력 스크롤하고 콘텐츠 키를 눌러 보류에 어 탭 제스처를 시뮬레이션 하기 위해 사용할 수 있습니다.
* **[블 룸](gestures.md#bloom) 또는 홈**: HoloLens 피우는 제스처 또는 셸에 반환 하 고 시스템 동작을 수행 하는 컨트롤러의 홈 단추를 사용 합니다.

실습에서는 다양 한 reprresentation HoloLens V2에 있습니다.  주행 제스처 추적/not 추적 및에 사용할 수 있을 뿐만 실습은 이제는 명확 하 고 기본 모델에 적합 하 고 개발자에 게 노출에 연결 합니다.  각 추적된 지점 20을 소개합니다.  
* **Joint**: 지정 된 추적된 손 모양에 대 한 20 개의 추적된 위치 중 하나입니다. 지정 해야이 연결 된 3d 공간입니다.
* **발생할**: 모든 추적에서 음의 전체 컬렉션입니다. 이때 20이 음의 컬렉션입니다. 

이때 각 공동 위치 에뮬레이터 UI 통해 개별적으로 직접 제어를 공개 하지 않습니다. 그러나 API 시뮬레이션을 통해 설정할 수 있습니다. 대신, 에뮬레이터를 전환할 수 있는 유용한 대표 포즈 집합이 있습니다.

또한 시뮬레이션 된 센서 입력의 상태를 제어할 수 있습니다.
* **재설정**: 모든 시뮬레이션 된 센서를 기본값으로 돌아갑니다.
* **추적**: 위치 추적 모드를 순환 합니다. 다음을 포함합니다.
  * **기본값**: OS는 시스템의 요청에 따라 최상의 추적 모드를 선택 합니다.
   * **방향**: 강제로 방향 으로만 이동 가능한 추적 시스템의 요청에 관계 없이 합니다.
   * **위치**: 시스템 요청에 관계 없이 강제로 위치 추적이 됩니다.

표시 방법 각각의 유형은 다음 표에서 입력 키보드, 마우스 및 Xbox 컨트롤러에 매핑됩니다. 각 형식에는 입력된 컨트롤 모드에 따라 서로 다른 매핑 입력된 컨트롤 모드에 대 한 자세한 내용은이 문서의 뒷부분에 제공 됩니다.

|  피치 |  위쪽/아래쪽 화살표 |  위쪽 / 아래쪽으로 끕니다. |  위쪽 / 아래쪽 오른쪽 엄지 스틱 | 
|  Roll |  Q / E |  |  DPad 왼쪽 / 오른쪽 | 
|  X |  A / D |  |  왼쪽 / 오른쪽 왼쪽된 엄지 스틱 | 
|  Y |  페이지 위로/아래로 페이지 |  |  DPad 위쪽 / 아래쪽 | 
|  Z |  W / S |  |  위쪽 / 아래쪽 왼쪽된 엄지 스틱 |

|  블 룸 |  F2 또는 Windows 키 (Windows 키만로 작동 HoloLens 에뮬레이터) |  |  B 버튼 |

|  다시 설정 |  Esc 키 |  |  시작 단추 | 
|  추적 |  T 또는 F3 |  |  X 버튼 |

## <a name="targeting"></a>대상 지정 

위의 입력된 개념 중 일부를 독자적으 합니다.  작업, Bloom, Reset 및 추적 완료 개념은 필요가 없습니다 및 모든 추가 한정자를 대상으로 하 여 영향을 받지 않습니다.  그러나 여러 대상 중 하나를 나머지 개념을 적용할 수 있습니다. 어떤 것이 명령에 적용 해야 하는 대상 의도 지정 하는 방법을 도입 했습니다.  모든 경우에 targtet 개체는 키보드 누름 또는 UI를 통해 지정 하는 것이 같습니다.  일부 경우에는 xbox 컨트롤러를 사용 하 여 직접 지정할 수 이기도 합니다. 

다음 표에서 대상 지정 및 각 활성화 방법에 대 한 옵션을 설명 합니다.

| Object | 키보드 한정자 | 컨트롤러 한정자 | 에뮬레이터 UI 한정자 |
|----------|----------|----------|----------|
| 본문 | <default> | <default> | <default> |
| 헤드 | 보류 시간 | <None available> | UI에서 Head 압정 |
| 왼쪽/컨트롤러 | 왼쪽 Alt 단추 | 왼쪽된 어깨 단추 | 왼쪽 위 압정 | 
| 오른쪽/컨트롤러 | 오른쪽 Alt 단추 | 올바른 자격 증명 입력 단추 | 오른쪽 위 압정 |
| 눈 | 보류 Y | <No contoller modifier available> | 눈 압정 |
  
다음 표에서 각 대상 한정자 각 코어 이동 입력 개념을 매핑하는 방법을 보여 줍니다.

|  기본 (본문) |  직접/컨트롤러 (alt 키를 상태 떠맡을 /) |  Head (보류 H)  |  눈 (보류 Y) |
|----------|----------|----------|----------|
|  Yaw |  왼쪽 / 오른쪽 본문 설정 |  왼쪽 / 오른쪽 직접 이동 |  왼쪽 / 오른쪽 헤드 설정 | 응시가 찾습니다 왼쪽/오른쪽 |
|  피치 |  위쪽 / 아래쪽 헤드 설정 |  직접을 위 / 아래로 이동 |  위쪽 / 아래쪽 헤드 설정 | 응시가 찾습니다 위쪽/아래쪽 | 
|  Roll |  왼쪽 / 오른쪽 롤포워드 헤드 |  |  왼쪽 / 오른쪽 롤포워드 헤드 | (작업 없음) |
|  X |  왼쪽 / 오른쪽 슬라이드 본문 |  왼쪽 / 오른쪽 위/컨트롤러 이동 |  왼쪽 / 오른쪽 헤드 설정 | (작업 없음) |
|  Y |  본문을 위 / 아래로 이동 |  직접/컨트롤러를 위 / 아래로 이동 |  위쪽 / 아래쪽 헤드 설정 | (작업 없음) |
|  Z |  본문 앞 / 뒤로 이동 |  직접/컨트롤러 앞 / 뒤로 이동 |  위쪽 / 아래쪽 헤드 설정 | (작업 없음) |
 
참고: Windows Mixed Reality 시뮬레이터만 컨트롤러 단추 한편 또는 다른 한정자를 대상으로 직접 사용 하 여 대상으로 지정 된 일 수 있습니다. 마찬가지로, HoloLens 에뮬레이터만에서 명확 하 고 직접 포즈 한편 또는 직접 한정자를 사용 하 여 대상으로 지정 된 수 있습니다. 
 
## <a name="controlling-an-app"></a>앱을 제어합니다.

이 문서에서는 입력된 형식 및 HoloLens 에뮬레이터와 Windows Mixed Reality 시뮬레이터에서 사용할 수 있는 입력된 모드의 전체 집합에 설명 했습니다. 다음과 같은 컨트롤 집합이 일상적으로 사용에 대 한 제안 됩니다.

|  연산 |  키보드 및 마우스 |  컨트롤러 | 
|----------|----------|----------|
|  본문 X |  A / D |  왼쪽 / 오른쪽 왼쪽된 엄지 스틱 | 
|  본문 Y |  페이지 위로/아래로 페이지 |  DPad 위쪽 / 아래쪽 | 
|  본문 Z |  W / S |  위쪽 / 아래쪽 왼쪽된 엄지 스틱 | 
|  본문 요 |  왼쪽 / 오른쪽 마우스를 끌기 |  왼쪽 / 오른쪽 오른쪽 엄지 스틱 | 
|  헤드 요 |  H + 끌기 마우스 왼쪽 / 오른쪽 |  (키보드)에 대 한 H + 오른쪽 엄지 스틱 왼쪽 / 오른쪽 | 
|  헤드 홍보 |  위 / 아래로 끌어 마우스 |  위쪽 / 아래쪽 오른쪽 엄지 스틱 |
