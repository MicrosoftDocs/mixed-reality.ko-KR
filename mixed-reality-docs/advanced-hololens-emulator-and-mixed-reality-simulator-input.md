# <a name="advanced-hololens-emulator-and-mixed-reality-simulator-input"></a>고급 HoloLens 에뮬레이터 및 혼합 현실 시뮬레이터 입력

대부분의 에뮬레이터 사용자는 [HoloLens 에뮬레이터](using-the-hololens-emulator.md#basic-emulator-input) 또는 [Windows Mixed Reality 시뮬레이터](using-the-windows-mixed-reality-simulator.md#basic-simulator-input)에 대 한 기본 입력 컨트롤을 사용 해야 합니다. 아래 세부 정보는 보다 복잡 한 유형의 입력을 시뮬레이트하는 데 필요한 고급 사용자를 위한 것입니다.


## <a name="concepts"></a>개념

HoloLens 에뮬레이터 및 Windows Mixed Reality 시뮬레이터에 대 한 가상 입력 제어를 시작 하려면 먼저 몇 가지 개념을 이해 해야 합니다.

동작은 장면에서 항목의 위치와 방향을 제어 하 고 변경 하는 것을 말합니다. 대상 제어 가능한 개체의 경우 움직임은 3 개의 축을 따라 회전 및 변환 (이동)을 통해 제어 됩니다.
* **요**: 왼쪽 또는 오른쪽으로 설정 합니다.
* **피치**: 설정 하거나 축소 합니다.
* **롤**: Roll side.
* **X**: 왼쪽 또는 오른쪽으로 이동 합니다.
* **Y**: 위아래로 이동 합니다.
* **Z**: 앞뒤로 이동 합니다.

[제스처](gestures.md) 및 동작 컨트롤러 입력은 실제 장치와 밀접 하 게 매핑됩니다.
* **작업**: 이는 컨트롤러의 동작 단추를 끌어올 때 엄지와 집게 손가락를 누르는 동작을 시뮬레이션 합니다. 예를 들어 작업 입력을 사용 하 여 공기 탭 제스처를 시뮬레이션 하 고, 콘텐츠를 스크롤하고, 길게 누르기를 수행할 수 있습니다.
* **[블 룸](gestures.md#bloom)/시스템 제스처 또는 Home**: HoloLens 블 룸/시스템 제스처 또는 컨트롤러의 홈 단추는 셸로 돌아가서 시스템 작업을 수행 하는 데 사용 됩니다.

Reprresentation에는 HoloLens 2가 풍부 하 게 포함 되어 있습니다.  이제는 추적/추적 하지 않고 제스처를 구동 하는 데 사용할 수 있을 뿐 아니라, 이제는이에 맞는 스 켈 레 톤 모델을 만들어 개발자에 게 노출 합니다.  이렇게 하면 각각 26 개의 추적 된 지점이 도입 됩니다.  
* **조인트**: 지정 된 추적 된 손에 대해 추적 된 위치 20 개 중 하나입니다. 그러면 점이 3d 공간에 연결 됩니다.
* **포즈**: 추적 된 손으로 이루어진 모든 조인트의 완전 한 컬렉션입니다. 이때이는 26 개 조인트의 컬렉션입니다. 

이번에는 사용자가 시뮬레이션 API를 통해 설정할 수 있지만 에뮬레이터 사용자 인터페이스를 통해 각 연결점의 직접 제어를 개별적으로 노출 하지 않습니다. 대신 에뮬레이터에서를 전환 하는 데 사용할 수 있는 일련의 유용한 담당자가 있습니다.

시뮬레이션 된 센서 입력의 상태를 제어할 수도 있습니다.
* **다시 설정**: 그러면 시뮬레이션 된 모든 센서가 기본값으로 반환 됩니다.  HoloLens 2 에뮬레이터부터 다시 설정의 범위는 적절 한 보조 키 또는 단추 (왼쪽 및/또는 오른쪽 Alt 또는 게임 패드의 왼쪽 및/또는 오른쪽 범퍼)를 사용 하 여 원하는 손을 사용 하 여 한 번 또는 두 번까지 지정할 수 있습니다.
* **추적**: 위치 추적 모드를 순환 합니다. 다음을 포함합니다.
  * **기본값**: OS는 시스템의 요청에 따라 가장 적합 한 추적 모드를 선택 합니다.
   * **방향**: 시스템에 대 한 요청에 관계 없이 방향 전용 추적을 강제 적용 합니다.
   * **위치**: 시스템에 대 한 요청에 관계 없이 위치 추적을 강제 적용 합니다.

## <a name="targeting"></a>대상 지정 

위의 입력 개념 중 일부는 고유 합니다.  Action, 블 룸/System, Reset 및 Tracking은 전체 개념 이며, 필요 하지 않으며, 대상 지정을 위한 추가 한정자의 영향을 받지 않습니다.  그러나 나머지 개념은 여러 대상 중 하나에 적용 될 수 있습니다. 명령을 적용 해야 하는 대상을 지정 하는 방법을 소개 했습니다.  모든 경우에 UI를 통해 또는 targtet 개체를 키보드 누름을 통해 지정할 수 있습니다.  경우에 따라 xbox 컨트롤러를 직접 지정 하는 것도 가능 합니다. 

다음 표에서는를 대상으로 하는 옵션과 각 항목을 활성화 하는 방법을 설명 합니다.

| Object | 키보드 한정자 | Controller 한정자 | 에뮬레이터 UI 한정자 |
|----------|----------|----------|----------|
| Body | (기본값) | (기본값) | (기본값) |
| 사장 | 보유 H | (사용할 수 없음) | (사용할 수 없음) |
| 왼쪽/컨트롤러 | 왼쪽 Alt 단추 길게 | 왼쪽 어깨 단추 누르기 | 왼쪽 압정 | 
| 오른쪽/컨트롤러 | 오른쪽으로 Alt 단추 길게 | 오른쪽 어깨 단추 누르기 | 오른쪽 압정 |
| 눈 | 유지 Y | (사용할 수 없음) | 눈 압정 |
  
다음 표에서는 각 대상 한정자가 각 핵심 이동 입력 개념을 매핑하는 방법을 보여 줍니다.

|  | 기본 (본문) |  손/컨트롤러 (Alt 키를 누른 채, 게임 패드 어깨 단추를 누르고 있거나 UI 압정 설정/해제) |  헤드 (길게)  |  눈 (Y 포함 또는 UI 압정 전환) |
|----------|----------|----------|----------|
|  Yaw |  왼쪽/오른쪽으로 본문 설정 |  왼쪽/오른쪽으로 왼쪽 이동 |  왼쪽/오른쪽으로 설정 | 아이 응시 왼쪽/오른쪽 모양 |
|  피치 |  위쪽/아래쪽으로 설정 |  손 모양 위로/아래로 이동 |  위쪽/아래쪽으로 설정 | 아이 응시에서 위/아래로 조회 | 
|  Roll |  왼쪽/오른쪽 롤 헤드 |  |  왼쪽/오른쪽 롤 헤드 | (작업 없음) |
|  X |  왼쪽/오른쪽으로 슬라이드 본문 |  손 모양/컨트롤러를 왼쪽/오른쪽으로 이동 |  왼쪽/오른쪽으로 설정 | (작업 없음) |
|  Y |  본문을 위/아래로 이동 |  손 모양/컨트롤러 위로/아래로 이동 |  위쪽/아래쪽으로 설정 | (작업 없음) |
|  Z |  본문 앞으로/뒤로 이동 |  손 모양/컨트롤러 앞으로/뒤로 이동 |  위쪽/아래쪽으로 설정 | (작업 없음) |
 
 
## <a name="controlling-an-app"></a>앱 제어

다음 컨트롤 집합은 일상적인 사용을 위해 제안 됩니다.

|  작업(Operation) |  키보드 및 마우스 |  컨트롤러 | 
|----------|----------|----------|
|  본문 X |  A/D |  왼쪽 엄지 스틱 왼쪽/오른쪽 | 
|  본문 Y |  Page up/page down |  DPad up/down | 
|  본문 Z |  W/S |  왼쪽 엄지 스틱 위쪽/아래쪽 | 
|  본문 요 |  마우스 왼쪽/오른쪽으로 끌기 |  오른쪽 엄지 스틱 왼쪽/오른쪽 | 
|  헤드 요 |  H + 마우스 왼쪽/오른쪽으로 끌기 |  H (키보드) + 오른쪽 엄지 스틱 왼쪽/오른쪽 | 
|  헤드 피치 |  마우스를 위아래로 끌기 |  오른쪽 엄지 스틱 위쪽/아래쪽 | 
|  헤드 롤 |  Q/E |  DPad left/right | 
|  손/컨트롤러 X |  Alt + A/D |  왼쪽/오른쪽 어깨 + 왼쪽 엄지 스틱 | 
|  손/컨트롤러 Y |  Alt + Page up/page down |  어깨 + DPad up/down | 
|  손/컨트롤러 Z |  Alt + W/S |  어깨 + 왼쪽 엄지 스틱 위쪽/아래쪽 | 
|  손/컨트롤러 요 |  Alt + 마우스 왼쪽/오른쪽으로 끌기 |  왼쪽/오른쪽 엄지 스틱 + 오른쪽 엄지 스틱 | 
|  손/컨트롤러 피치 |  Alt + 마우스를 위로/아래로 끌기 |  어깨 + 오른쪽 엄지 스틱 위쪽/아래쪽 | 
|  손/컨트롤러 롤 |  Alt + Q/E |  왼쪽/오른쪽 어깨 + DPad | 
|  Action |  마우스 오른쪽 단추 |  트리거되도록 | 
|  블 룸/시스템/홈 |  F2 또는 Windows 키 |  B 버튼 | 
|  다시 설정 |  이스케이프 |  시작 단추 | 
|  추적 |  T |  X 버튼 | 
|  스크롤 |  Alt + 마우스 오른쪽 단추 + 마우스를 위로/아래로 끌기 |  어깨 + 트리거 + 오른쪽 엄지 스틱 위쪽/아래쪽 | 
|  더 빠르게 이동/회전 | 왼쪽 또는 오른쪽 Shift 키 | 오른쪽 엄지 스틱을 누르고 있습니다. |
|  느리게 이동/회전 | 왼쪽 또는 오른쪽 Ctrl 키 | 왼쪽 엄지 스틱을 누르고 있습니다. |

| 작업(Operation) | 바로 가기 | 설명/참고 사항 |
|-----------|----------|-------------|
