# <a name="release-notes---october-2017"></a>릴리스 정보-2017 년 10 월

혼합 현실을 Windows 시작! 릴리스를 **[Windows 10 Fall Creators Update](https://blogs.windows.com/windowsexperience/2017/10/17/whats-new-windows-10-fall-creators-update/)** new에 대 한 지원이 도입 되었습니다 [Windows Mixed Reality 몰입 형 헤드셋](immersive-headset-hardware-details.md) 고 [컨트롤러 동작 ](motion-controllers.md), VR 게임을 플레이 새로운 세계를 탐색 하 활성화 및 몰입 형 엔터테인먼트 연결할 때 경험을 [Windows Mixed Reality capable PC](https://docs.microsoft.com/windows/mixed-reality/enthusiast-guide/windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines)합니다.

Windows Mixed Reality 헤드셋 및 동작 컨트롤러의 해제는 대규모 팀의 노력 및에 대 한 주요 단계 앞으로 작업을 합니다 [Windows Mixed Reality 플랫폼](mixed-reality.md)를 포함 하는 [Microsoft HoloLens](hololens-hardware-details.md). HoloLens 하지 릴리스는 Windows 10 Fall Creators Update를 사용 하 여 업데이트를 받을 때 알고는 HoloLens에 작업을 중지 하지 않은; 많은 Windows Mixed Reality를 전체적으로 적용에서 최근 작업에 대 한 정보 및 학습 해야 합니다. 사실 Windows Mixed Reality 몰입 형 헤드셋 및 컨트롤러 유용한 항목을 나타내는 동작 HoloLens 개발 너무 도구, 동일한 Api를 가리키고 개념 둘 다에 적용 합니다.

각 장치에 대 한 최신 릴리스를 업데이트 하려면 열고 합니다 **설정** 앱으로 이동 **업데이트 및 보안**을 선택한 후는 **업데이트 확인** 단추입니다. Windows 10 PC에 수동으로 설치할 수 사용 하 여 Windows 10 Fall Creators Update 합니다 [Windows 미디어 작성 도구](https://www.microsoft.com/software-download/windows10)합니다.

* **[넓은 Pc 범위에 대 한 지원](https://docs.microsoft.com/windows/mixed-reality/enthusiast-guide/windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines)**  -Windows Mixed Reality 그 어느 때 보다 데스크톱 VR 환경을 더 많은 사람들이 사용 하면, 선택에 대 한 지원을 통합 그래픽 카드 및 Pc에서 값인 499 USD를 시작 합니다.

* **[VR 게임 및 Microsoft Store 앱의 놀라운](https://www.microsoft.com/store/collections/MR-All-ImmersiveContent/)**  -몰입 형 엔터테인먼트 등 SUPERHOT VR Hulu VR 및 360 장대한 게임 비디오 및 애리조나 햇빛과, Microsoft Store Windows 혼합 환경을 콘텐츠의 범위와 같은 현실입니다.

## <a name="known-issues"></a>알려진 문제

훌륭한 Windows Mixed Reality 환경을 제공 하기 위해 노력 했습니다 있지만 몇 가지 알려진된 문제가 여전히 추적 하는 것입니다. 다른 사용자가 있다면 하세요 [의견](give-us-feedback.md)합니다.

### <a name="desktop-app-in-the-windows-mixed-reality-home"></a>Windows Mixed Reality 홈에서 데스크톱 앱
* 캡처 도구는 데스크톱 앱에서 작동 하지 않습니다.
* 데스크톱 앱에 다시 시작할 때 설정이 유지 되지 않습니다.
* 혼합 현실 포털 미리 보기를 사용 하 여 Windows Mixed Reality 홈에서 데스크톱 앱을 열 때 바탕 화면에, 무한 미러 효과 확인할 수 있습니다. 
* 데스크톱 앱을 실행 문제를 일으킬 수 성능에 비-Ultra Windows 혼합 현실 Pc; 권장 되지 않습니다.  
* 데스크톱 앱 수 자동으로 시작은 보이지 않는 창을 바탕 화면에서 포커스를가지고 때문입니다. 
* 데스크톱 사용자 계정 컨트롤 프롬프트 헤드셋 프롬프트 완료 될 때까지 검정색을 표시 하 게 합니다.

### <a name="windows-mixed-reality-setup"></a>Windows Mixed Reality 설치
* 연결 된 헤드셋을 사용 하 여 Windows를 설정할 때 PC 모니터 빈 이동할 수 있습니다. Windows 설치를 완료 하려면 PC 모니터에 대 한 출력을 사용 하려면 헤드셋을 분리 합니다.
* 경계를 만들 때 추적 실패할 수 있습니다. 그렇다면 다시 시도 처럼 시스템 시간에 따른 공간에 자세히 설명 합니다.
* Windows Mixed Reality 설치 중에 켜거나 Cortana를 설정한 경우이 변경은 데스크톱 Cortana 설정에 적용 됩니다.
* 연결 헤드폰 없으면 홈 Windows Mixed Reality를 처음 방문 하면 추가 팁을 놓칠 수 있습니다.
* Bluetooth 헤드폰 컨트롤러 동작에 방해가 될 수 있습니다. Windows Mixed Reality 세션 동안 Bluetooth 컨트롤러 전원 또는 연결 해제는 것이 좋습니다.

### <a name="games-and-apps-from-windows-store"></a>Windows 스토어에서 앱 및 게임
* Forza Motorsports 6과 같은 일부 그래픽 위주 게임을 Windows Mixed Reality 내에서 재생 하는 경우 낮은 사양의 Pc에서 성능 문제가 발생할 수 있습니다.

### <a name="audio"></a>오디오
* 위에서 언급 했 듯이 Bluetooth 오디오 주변 장치 Windows Mixed Reality 음성 및 공간 사운드 환경을 잘 작동 하지 않습니다. 동작 컨트롤러 환경을는 영향을 저하 수 있습니다. Windows Mixed Reality Bluetooth 오디오 헤드셋을 사용 하는 것은 좋지 않습니다.
* 오디오 장치에 연결 (또는 일부)를 사용할 수 없습니다 헤드셋 장치는 하지 착용 하는 경우 오디오 재생 합니다. 하나의 오디오 헤드셋만 있는 경우에 오디오 헤드셋 헤드셋 대신 호스트 PC에 연결 하는 것이 좋습니다. 따라서 다음 기능을 해제 해야 "헤드셋 오디오도 전환" 경우 **설정을** > **혼합 현실** > **오디오 및 음성**합니다.
* 여러 SteamVR를 통해 시작 등 일부 응용 프로그램 오디오를 손실 하거나 시작 또는 중지 혼합 현실 포털 오디오 장치 변경 될 때 중단 수 있습니다. 이 문제를 해결 하려면 혼합 현실 포털 앱을 연 후 앱을 다시 시작 합니다.
* Windows Mixed Reality 헤드셋을 사용 하기 전에 호스트 PC에서 사용 하도록 설정 하는 Cortana가 Windows Mixed Reality 홈 주위 배치 앱에 적용할 공간 사운드 시뮬레이션을 손실 될 수 있습니다. 해결 방법은 PC도 헤드셋에 연결 된 오디오 장치에 연결 된 모든 오디오 장치에서 "Windows Sonic에 대 한 헤드폰"를 사용 하도록 설정 하려면 것:
   1. 바탕 화면 작업 표시줄에서 화자 아이콘을 마우스 왼쪽 단추로 클릭 하 고 오디오 장치 목록에서 선택 합니다.
   2. 바탕 화면 작업 표시줄에서 화자 아이콘을 마우스 오른쪽 단추로 클릭 하 고 "스피커 setup" 메뉴에서 "Windows Sonic에 대 한 헤드폰"를 선택 합니다.
   3. 모든 오디오 장치 (끝점)에 대해 이러한 단계를 반복 합니다.
>[!NOTE]
> - Windows Mixed Reality 홈 오디오 장치에이 설정을 적용 하려면에서 바탕 화면 앱 창을 내에서이 작업을 수행 해야 하므로 헤드폰/발표자 헤드셋에 연결 된 해당 입고 있는 경우가 아니면 표시 되지 헤드셋에 연결 (또는 통합 헤드셋)에.
> - "Hey Cortana 통해 Cortana 대응"를 해제 하려면 다른 옵션은 **설정을** > **Cortana** Windows Mixed Reality를 시작 하기 전에 바탕 화면에 있습니다.

* Windows Mixed Reality 헤드셋와 같은 USB 허브 (외부 또는 내부 PC)를 공유 하는 다른 멀티미디어 USB 장치 (예: 웹 캠), 드문 경우에서 헤드셋의 오디오 잭이/헤드폰 있을 수 있습니다 하거나 버 징 사운드 또는 오디오 없는 모든. 다른 장치와 같은 허브를 공유 하거나 하지 다른 USB 멀티미디어 장치 연결 끊기/해제 하는 USB 포트에 헤드셋 하 여 해결할 수 있습니다.
* 매우 드문 경우에는 호스트 PC의 USB 허브 Windows Mixed Reality 헤드셋 데 필요한 기능 부족을 제공할 수 없습니다 및 헤드셋 연결할 헤드폰 방해 폭주 표시 될 수도 있습니다.

### <a name="speech"></a>음성 명령
* Cortana가 오디오 신호 명령에 응답을 수신 하 고 사고 및 오디오를 재생 하는 데 실패할 수 있습니다.
* 중국 및 일본 시장에서 Cortana 표시 안 함 올바르게 텍스트 Cortana 원을 아래 사용 중입니다.
* Cortana 처음으로 혼합 현실 포털 세션에서 호출 되는 그녀가 느려질 수 있습니다. 해결할 수 있습니다이 있는지 "Let Cortana Hey Cortana에 대응" 하 여 아래 **설정을** > **Cortana** > **Cortana에 대해** 는 사용할 수 있습니다.
* Cortana는 Windows Mixed Reality Ultra Pc 되지 않는 Pc에 느려질 수 있습니다.
* Wi-fi 연결 사용 하지 않기 때문에 작동 하지 않는 받아쓰기에 대 한 오류 대화 상자가 시스템 키보드에서 Windows Mixed Reality UI 언어와 다른 언어로 설정 되 면 Windows Mixed Reality에서 키보드에서 받아쓰기를 사용 하 여 발생 합니다. 해결 하려면 문제는 단순히 시스템 키보드 언어가 Windows Mixed Reality UI 언어와 일치 있는지를 확인 합니다.
* 스페인은 Windows Mixed Reality에 대 한 음성이 사용 되는 시장으로 제대로 인식 되지 됩니다.

### <a name="holograms"></a>홀로그램
* 홈에 Windows Mixed Reality에 많은 수의 홀로그램 배치한, 경우 일부 수 사라지고 분야를 살펴보면 대로 다시 나타납니다. 이 방지 하려면 해당 영역의 홈 Windows Mixed Reality를 홀로그램 중 일부를 제거 합니다.

### <a name="motion-controllers"></a>컨트롤러 동작
* 경우에 따라에 지에서 웹 페이지를 클릭 하면 콘텐츠를 클릭 하는 대신 확대 됩니다.
* 가장자리에 있는 링크를 클릭 하는 경우에 따라 선택이 작동 하지 않습니다.
